Mit dem Veristore-Plugin bindest Du ganz einfach Kundenbewertungen in Deine Seite ein. Als hoverndes Widget am Seitenrand, als Badge auf der Seite und als strukturierte Daten zur Anzeige der Sterne in der Google Suche.

== Description ==

Trust-Signals und Kundenbewertungen im bekannten Sternchen-Format steigern nachweislich die Klickzahlen bei Google, die Konversionsraten, die Warenkorbgröße und insgesamt den Umsatz in Deinem Onlineshop. Das Veristore-Bewertungssystem basiert auf jahrelanger Erfahrung im E-Commerce. Wir haben die Wünsche von Onlineshopbetreibern und Endkunden gleichermaßen berücksichtigt.

Mit dem Veristore Plugin bindest Du ganz einfach Kundenbewertungen in Dein Wordpress ein. Als hoverndes Widget am Seitenrand, als Badge auf der Seite und als strukturierte Daten zur Anzeige der Sterne in der Google Suche.


== Installation ==

Die Installation ist in vier Schritten erledigt:

1. Lade das Plugin hoch. Dafür gibt es zwei Möglichkeiten:
	a. Entpacke das veristore.zip und lade das Verzeichnis veristore per FTP nach /wp-content/plugins hoch.
	b. oder gehe im Wordpress Backend auf "Plugins -> Installieren" und klicke auf dort auf den Button "Plugin hochladen" und wähle das veristore.zip aus.
	c. oder gehe im Wordpress Backend auf "Plugins -> Installieren" und tippe in das Suchfeld "veristore" ein. Dann klickst Du auf "installieren".
2. Aktiviere jetzt das Plugin über das Wordpress-Menü "Plugins".
3. Gehe im Wordpress-Menü "Plugins" beim Verstore-Plugin auf "Einstellungen" und gib dort Dein Veristore-Token ein. Wenn Du noch kein Token hast, musst Du Dich unter www.veristore.de registrieren.
4. Gehe in das Menu "Design - Widgets" und schiebe dort die Widgets "Veristore Badge" und "Veristore strukturierte Daten" von der Spalte "Verfügbare Widgets" in die Box "Unterhalb des Inhalts".

== Frequently Asked Questions ==

= Was ist das Veristore Widget =

Das Veristore Widget ist ein Fähnchen, das am Seitenrand angezeigt wird. Du kannst zwischen zwei Größen wählen und den Abstand zum unteren Bildschirmrand einstellen.

= Was ist das Verstore Badge? =

Mit dem integrierten Code-Generator kannst Du eine Box erstellen, in der Dein Veristore-Siegel, die durchschnittliche Bewertung und der letzte Kommentar angezeigt werden. Diese Box kannst Du dann an beliebiger Stelle auf Deiner Seite einfügen. Die Daten werden einmal am Tag aktualisiert.

= Werden die Sterne auch in der Google-Suche angezeigt? =

Ja. :)

= Ist Veristore kostenfrei? =

Ja. Es gibt verschiedene Tarife mit unterschiedlichen Optionen. In der Grundfunktion (Veristore Starter) ist das Bewertungssystem kostenfrei und voll funktionstüchtig.

= Wie lange dauert die Prüfung meiner Webseite bis zur Freischaltung? =

In der Regel erfolgt die Freischaltung innerhalb weniger Stunden. In seltenen Fällen können auch schon mal 48 Stunden vergehen.
